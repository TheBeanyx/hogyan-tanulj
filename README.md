## 🚀 Telepítés és használat

Kövesd az alábbi lépéseket a projekt futtatásához:

1.  **Klónozd a tárolót:**
    ```bash
    git clone [https://github.com/felhasznalonev/projekt-neve.git](https://github.com/felhasznalonev/projekt-neve.git)
    ```
2.  **Lépj be a mappába:**
    ```bash
    cd projekt-neve
    ```
3.  **Telepítsd a függőségeket:**
    ```bash
    npm install  # vagy pip install -r requirements.txt
    ```
4.  **Indítsd el:**
    ```bash
    npm start    # vagy python main.py
    ```
